# ecampus-

## 前言 
為了解決教學平台無法自動適屏各個裝置，我們決定重新改良教學平台的種種缺點！
<br>
但因為我們是以爬蟲方式爬取教學平台的資訊，也就是說我們的產品還是基於**教學平台**。  
我們主要想實現快速、直覺的操作，拒絕繁瑣以及煩人的跳出視窗！

使用工具、語言
------------
#### 後端
* Python 3
  * BeautifulSoup
  * requests
* Django
* Sqlite 3
#### 前端
* Bootstrap 4
* Javascript
* Html + Css

已實現功能
---------
可以用學號登入到教學平台，並且列出所有課程。
